## Video chat
Simple video chat with livekit.

![alt videochat image](https://github.com/Qraree/assets/blob/main/chat.jpg)

### Stack:
- Livekit
- Socket.io
- Express
- Next.js 13
- Typescript
- tailwind

### Deploy
- Ubuntu 22.04
- docker-compose
- nginx-proxy
- ssl certificate with let's encrypt
- CI/CD with Github actions

[website link](https://dddsqrt-videochat.ru)
